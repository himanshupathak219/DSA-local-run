# DSA Local Run

This repository contains Data Structures and Algorithms (DSA) learning projects implemented in C++. It's a collection of fundamental algorithms and data structure implementations organized by topic.

## 📁 Folder Structure

### **arrays/**
Contains programs related to array operations and manipulations:
- Linear search implementation
- Finding largest and smallest elements
- Reversing arrays
- Sum and product calculations
- Swapping max and min elements
- Printing unique values in arrays
- Finding intersections between two arrays

### **binarynumbersystem/**
Binary number system and base conversion programs:
- Binary to decimal conversion
- Prime number generation from 1 to N

### **bitwiseoperator/**
Bitwise operations and bit manipulation:
- Basic bitwise operations
- Reversing integers using bit manipulation

### **conditional statement and loops/**
Control flow and iteration programs:
- Nested loops
- Prime number checking
- Sum of odd numbers
- Loop variations and patterns

### **functions/**
Function implementation and recursion examples:
- Fibonacci series
- Binomial coefficients
- Prime number generation
- Sum calculations
- Min/Max functions

### **loops/**
Loop fundamentals and iterative programs

### **VECTORS/**
Vector data structure operations:
- Vector basics and operations
- Intersection of two vectors
- Single number problem (LeetCode style)

## 🛠️ Technologies Used
- **Language:** C++
- **Compiler:** GCC (MinGW)
- **IDE:** Visual Studio Code

## 📝 Purpose
This repository serves as a learning resource for understanding fundamental DSA concepts and implementing them from scratch in C++.

## 📦 Build & Run
Individual programs can be compiled and executed using the GCC compiler:
```bash
g++ -o program_name program_name.cpp
./program_name
```

Or use the VS Code build task to compile the active file.
